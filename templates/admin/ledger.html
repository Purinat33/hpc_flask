Quick workflow (month-end)

  1. Set window and ensure the target period is Open.
  2. Accrue revenue in service month (Dr 1150 / Cr 4000).
  3. Issue invoice: Dr 1100; Cr 1150; Cr 2100 (if VAT enabled).
  4. Record cash: Dr 1000; Cr 1100 when paid.
  5. Run ECL (simplified approach; provision matrix by ageing) for A/R and contract assets.
  6. Close period (P&L ‚Üí 3000). If needed, re-open via dated reversal.
  7. Export the Posted GL (ZIP: CSV + manifest + HMAC) to downstream systems/e-Tax gateways.
